A Brief Overview of WebDAV


What is WebDAV?

WebDAV (Web Distributed Authoring Versioning) allows users to manipulate files on WWW servers directly from your computer to be published on the internet. WebDAV is separated into either class 1 or class 2. Class 1 is associated with editing, moving, creating, or deleting files and/or folders. Class 2 has the oppposite function, which can lock files and folders to prevent users from making any type of modifications to them.

WebDAV has both it's benefits as well as disadvangtages. In the following sections, we will explore more about both sides.


Benefits

  • Allows users to become more interactive with the web
  • This becomes a more convenient approach
  • Easily allow files to be uploaded to a web server
  • Users are able to lock files which can prevent confusion by only allowing one person to work on files
  • May be a more efficient method for teams to collaborate.

For more information about advantages of WebDAV, please visit Who Is Hosting This


Disadvantages

  • WebDAV is slower than FTP.
  • Connection may not be secure
  • Connection may become unstable
  • May not be able to handle large files
  • Some files may only be opened in a view mode, such as Microsoft Word files
